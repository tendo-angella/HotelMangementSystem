# 🏨 ParkEase: Hotel Management System

An integrated management system built with **Python**, **Django**, and **Bootstrap 5**. This system allows hotel staff to manage room availability, register guests, and handle service bookings through a clean, minimalist dashboard.

## ✨ Key Features
* **Room Management:** Categorize rooms (Single, Double, Suite) and track real-time occupancy.
* **Guest Registration:** Securely store guest profiles with ID and contact validation.
* **Automated Logic:** Automatic price calculation and room status updates upon guest checkout.
* **Zero-JS Approach:** Built using pure Django template logic and CSS for high performance and stability.

## 🛠️ Technology Stack
* **Backend:** Python 3.14 / Django 6.0
* **Frontend:** HTML5, Bootstrap 5 (Custom CSS)
* **Database:** SQLite (Development)
